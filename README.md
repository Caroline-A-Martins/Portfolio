# Meu Website de Portfólio

Desenvolvi este site de portfólio com o objetivo de exibir minha habilidade em criar e desenvolver aplicações web utilizando tecnologias modernas de desenvolvimento. O tutorial disponível no YouTube ofereceu valiosas perspectivas e direcionamento ao longo do processo de criação.

## Recursos
- Site Pessoal de Portfólio Responsivo usando HTML, CSS e JavaScript
- Rolagem suave em cada seção
- Inclui modos claro e escuro.
- Desenvolvido inicialmente com a metodologia Mobile First, em seguida para desktop.
- Compatível com todos os dispositivos móveis.
- Interface visualmente atraente e amigável ao usuário.
- Seções para exibir meu trabalho, habilidades e experiências
